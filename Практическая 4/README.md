Рекурсивный случай: возвращает последнюю цифру + результат для оставшейся части

Использует операцию % для получения последней цифры

Использует / для удаления последней цифры

2.3 Принцип работы Python
Функция принимает целое число

Использует целочисленное деление //

Базовый случай: n == 0

3. Анализ кода

Временная сложность: O(log₁₀n)

5. Ответ на контрольный вопрос
Разница между DFS и BFS:

DFS (Depth-First Search):

Обход в глубину

Использует стек (явный или неявный)

Временная сложность: O(V + E)

Пространственная сложность: O(V)

Подходит для поиска в глубоких деревьях

BFS (Breadth-First Search):

Обход в ширину

Использует очередь

Временная сложность: O(V + E)

Пространственная сложность: O(V)

Находит кратчайший путь в невзвешенном графе

Основное различие: DFS идет вглубь, BFS исследует все соседние вершины на текущем уровне.
